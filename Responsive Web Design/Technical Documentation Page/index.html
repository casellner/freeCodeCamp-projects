Introduction

This documentation does not cover every aspect of C++ and it is not the most in-depth documentation. However, it is a good reminder of syntax for the most common parts of C++.

Variables

Variables are pieces of data in memory. They are declared with a data type, a name, and (optionally) an initial value.

int number = 7; char letter = 'a'; bool boolean = true;
Input and Output

To use the input and output streams explained in this documentation, you must include the following lines at the top of your code:

#include <iostream> using namespace std;

You can then print to the console with the keyword cout, two less than signs (<<), and a text surrounded by double quotes ("")

cout << "Hello world!";

You can get input from the console with cin, two greater than signs (>>), and a variable. The following is a simple program which gets a character from a user:

char letter; cout << "Enter a letter: "; cin >> letter; cout << "You entered " << letter;
Math

The five arithmetic operations in C++ are:

  • addition with +
  • subtraction with -
  • multiplication with *
  • division with /
  • modulo (remainer from division) with %
int num1 = 7 + 3; int num2 = num1 - 4; int result = num1 * num2;

Additional math functions can be used with the cmath library. (See the next section for more information on functions.)

#include <cmath> cos(num1); //cosine log(num2); //logarithm
Functions

Functions break a program into smaller, more manageable parts. Functions may slightly decrease the performance of a program, but they greatly increase readability. The following code demonstrates a function which multiplies an integer by two.

#include <iostream> using namespace std; int mul2 (int num) {   num = num * 2;   return num; } int main() {   int input, result;   cout << "Enter an integer: ";   cin >> input;   result = mul2(input);   cout << "The result is " << result; }
If-Else Statements

If-Else statements control how a program behaves based on the value of a variable.

if (x == 5) {   cout << "x is 5"; } else {   cout << "x is not 5"; }
Loops

The most important C++ loops are the for loop and while loop. The for loop executes a specific number of times, and the while loop executes as long as its condition is true. The following loops print the numbers 1-5 and 6-10 respectively

for (int i = 1; i < 6; i++) {   cout << i; } int j = 6; while (j < 11) {   cout << j;   j++; }
Arrays

An array is a series of elements. While a variable can be visualized as a car, an array is more like a caravan.

You can declare an array of five characters with:

char array[5];

In C++, array elements start at 0. To set the first element of your array to a and the second element to b write:

array[0] = 'a'; array[1] = 'b';
Structures

Structures group related data together. For example: a number of rooms and vacancy status can be grouped into a House structure.

struct House {   int rooms = 2;   bool vacant = false; }
Classes

Classes, like structures, group related data, but classes also have their own functions. Private members are only accessible within the class, but public members are accessible wherever the class can be used.

class Cat {   private:     string name;     int hunger;   public:     feedCat(){hunger--;} }
Pointers

Variables store data, but pointers store a variable's location in memory. Pointer declarations require an asterisk (*), assignment requires an ampersand (&), and the value being pointed to can be dereferenced with an asterisk.

int num = 7; int *ptr; ptr = &num; cout << *ptr;
